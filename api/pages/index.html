Just an experiment for Roblox itself just for fun

How to use this?

Send a api request at /sendMessage endpoint with following format: 

{
    "username": "RobloxUser123"
    "content": "Hello World",
    "experience_id": "1234567890"
}

And it will be sent to the database accordingly.

Please note that only use if you're using via Studio (e.g. Roblox places/experience), never abuse with the api other than that
